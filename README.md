# AdventOfCode

Advent of Code Solutions written in TypeScript

## Requirements

- bun - https://bun.sh/

## Initializing the code for a certain day

```bash
cd 2024
bun init:day Day 1: Historian Hysteria
```

## Running the code

```bash
cd 2024
bun install
bun test 01 # or any other day number
```
